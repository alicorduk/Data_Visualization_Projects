# North American Causes of Death Analysis

This project provides a detailed analysis using various data visualization techniques to uncover trends in causes of death in North America from 1990 to 2019.

## Contents
- Introduction and motivation
- Data set description
- Analysis techniques
- Results and insights
- Limitations and conclusions

## Usage
1. Clone the repository
2. Navigate to the `North_American_Causes_of_Death_Analysis` folder
3. Open the `North American Causes of Death Analysis.docx` file
4. Review the analysis and insights

## Requirements
- Microsoft Word for reading the document
- Tableau and Excel for exploring the associated visualizations
