# Report in Power BI

This project features a comprehensive Power BI report with various data visualizations to present insights and trends effectively.

## Contents
- Introduction to the dataset
- Data transformations
- Visualization techniques
- Insights and trends

## Usage
1. Clone the repository
2. Navigate to the `Report_in_Power_BI` folder
3. Open the `Report-in-PowerBI.pbix` file in Power BI
4. Explore the report and interact with the visuals
