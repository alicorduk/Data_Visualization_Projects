# Building a Dashboard (Tableau)

This project walks through the process of building an interactive dashboard in Tableau.

## Contents
- Introduction to the dataset
- Dashboard building techniques
- Insights and trends

## Usage
1. Clone the repository
2. Navigate to the `Building_a_Dashboard` folder
3. Open the `Building a Dashboard.twbx` file in Tableau
4. Explore the workbook and interact with the visuals
