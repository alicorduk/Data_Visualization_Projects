# Aggregating Dimensions (Tableau)

This project focuses on aggregating dimensions in Tableau to analyze data more effectively.

## Contents
- Introduction to the dataset
- Aggregating dimensions techniques
- Insights and trends

## Usage
1. Clone the repository
2. Navigate to the `Aggregating_Dimensions` folder
3. Open the `Aggregating_Dimensions.twbx` file in Tableau
4. Explore the workbook and interact with the visuals
