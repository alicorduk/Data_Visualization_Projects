# Makeover Monday (Tableau)

This project re-visualizes the cost of mobile data by country using geographic visualization and color coding in Tableau.

## Contents
- Introduction to the dataset
- Visualization techniques
- Insights and trends

## Usage
1. Clone the repository
2. Navigate to the `Makeover_Monday` folder
3. Open the `Makeover_MuhammedCORDUK.twbx` file in Tableau
4. Explore the workbook and interact with the visuals
